# Neutrino

Please read documentation at:

[https://neutrinocms.com/docs](https://neutrinocms.com/docs)

## Requirements

- PHP 7.2+
- Laravel 6+
- Database

## Installation

Install package via composer:

```
composer require newelement/neutrino
```
